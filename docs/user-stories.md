# JeepneyX Phase 1A User Stories

## Corporate Visitor Stories
1. As a potential partner, I want to quickly understand what Jeepney X does so I can assess if a business discussion is relevant.
2. As an investor or stakeholder, I want to see a credible and modern company presentation so I can trust the brand.
3. As a transport operator, I want to view core capabilities so I can identify services related to EV operations.

## Inquiry Stories
1. As an interested client, I want to submit an inquiry form with my contact details and message so Jeepney X can contact me.
2. As a visitor, I want to see contact details clearly so I can reach Jeepney X directly by phone or email.
3. As a privacy-conscious user, I want to see consent text before submission so I understand follow-up expectations.

## Usability Stories
1. As a mobile user, I want touch-friendly navigation and readable content so I can browse without friction.
2. As a first-time visitor, I want a clear homepage hierarchy so I can understand the company in under a minute.
3. As a returning visitor, I want consistent header/footer links so I can move between pages quickly.
