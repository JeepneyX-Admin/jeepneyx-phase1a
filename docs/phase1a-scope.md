# JeepneyX Phase 1A Scope

## Objective
Deliver a static, professional corporate website for Jeepney X that establishes digital credibility for expo and business presentation while enabling direct inquiry capture.

## In Scope
- Static Home, About, and Contact pages
- Premium automotive-inspired homepage with full-width, full-screen hero
- Brand-aligned copy grounded in approved project documents
- Home section for capabilities (no separate capabilities page)
- Inquiry form using Formspree
- Responsive layout for desktop, tablet, and mobile
- GitHub Pages deployment readiness

## Out of Scope
- Backend, database, CMS, admin panel, authentication
- Laravel implementation (reserved for future phases)
- RFQ logic, quote engine, booking, dealer workflows
- E-commerce, payment gateway, product filtering system
- Advanced API workflows or runtime server dependencies

## Non-Negotiable UI Rule
The homepage hero must present as full browser width and full browser height, with strong imagery and readable overlaid messaging.

## Content Guardrails
- Use only approved content and verified facts
- Avoid invented claims, unsupported metrics, or unverified certifications
- Keep tone professional, concise, and investor-friendly

## Hosting Constraint
The website must run as a static site on GitHub Pages with no server runtime requirement.
